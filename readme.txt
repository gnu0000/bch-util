
Various productivity tools used while at BCH for managing survey data, etc...

AddLanguage.pl ............... This utility is for adding a new language or backfilling an existing language
AddResponseTranslations.pl ... This utility copies response translations from the demo to production DB
AddUIFields.pl ............... Use google translate to add another language UI
AnonymizeDB.pl ............... This utility anonymizes users names in the DB
CleanPhoneNumbers.pl ......... This utility cleans up crappy phone #'s 
CompareDBs.pl ................ Compare module IDs between 2 databases
CompareQuestions.pl .......... Compare the questions of a module from 2 databases
CompareUIFields.pl ........... Compare the uifields and langui from 2 databases
DumpDuplicates.pl ............ Displays Duplicate uifield/question/response Text 
FindText.pl .................. Find and dump Trivox Text Fields
FindVars.pl .................. Find <variables> in questiontext and responsetext fields
FixBrokenGMIs.pl ............. fix broken groupmembershipinclusions
FixInteractions.pl ........... fix broken groupids in interactionTemplates
fixrebel.pl .................. Change the drive letters in rebel.xml files to work with the current environment
GenerateModuleIndicies.pl .... Generate module indexes for the web based editor
GenModuleQuestions.pl ........ Generate question indexes for the web based editor
IDDumper.pl .................. Utility to dump Trivox Set or Module IDs
KillUser.pl .................. Eradicate a Trivox user from existence!
Latin1ToUTF8.pl .............. Convert text from Latin1 to utf8
MissingTextScan.pl ........... Scan Trivox metadata for missing text data
QuestionFixFirst.pl .......... Initial utility for cleaning up questiontext
QuestionFixLast.pl ........... Final utility for cleaning up questiontext
QuestionFixMiddle.pl ......... Middle utility for cleaning up questiontext
QuestionLinker.pl ............ Build links to the old web translator.
QuestionView.pl .............. Dump TriVox question string data all languages
ResponseFixFirst.pl .......... Initial utility for cleaning up responsetext
ResponseFixLast.pl ........... Final utility for cleaning up responsetext
ResponseView.pl .............. Dump TriVox response string data all languages
SurveyModuleList.pl .......... Dumps all survey/module info
UifieldFixFirst.pl ........... Initial utility for cleaning up uifields
UifieldFixLast.pl ............ Final utility for cleaning up uifields
UiFieldMissing.pl ............ Examnines uifields for missing text or translations
UifieldView.pl ............... Dump TriVox question string data all languages
